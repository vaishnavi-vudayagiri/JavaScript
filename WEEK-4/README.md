## Files Included

### 1. webpage.html
A basic HTML webpage demonstrating:
- Paragraphs
- Bold and Italic text formatting
- Headings 
- Hyperlinks 
- Unordered Lists

### 2. Nested_Unordered_List.html
An HTML file that demonstrates:
- Nested unordered lists
- Categories such as Fruits and Vegetables
- Sub-items inside each category

### 3. Pochi_The_Cat/index.html
A profile webpage for Pochi the Cat containing:
- Introduction section
- Image
- Ordered list of favorite food, hobbies, and hidden talent
- Useful links
- Last updated information

## Concepts Practiced
- HTML document structure
- Meta tags
- Headings and paragraphs
- Text formatting
- Links
- Images
- Ordered and unordered lists
- Nested lists

## How to Run
1. Download or clone this repository.
2. Open any `.html` file in a web browser.
3. View the rendered webpage.

## Learning Outcome
By completing these exercises, I learned how to:
- Build basic HTML pages
- Organize content using lists
- Add images and hyperlinks
- Structure web pages properly
